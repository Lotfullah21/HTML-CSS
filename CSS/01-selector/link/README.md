## Links

When adding styles for links, remember this order `(LVHA) -> link, visited, hover, active`.

```css
/* Links by default */
a:link {
	color: blue;
}
/* Links is visited by a user. */
a:visited {
	color: rgb(16, 35, 16);
}
/* Links is hovered by a mouse. */
a:hover {
	color: rgb(113, 188, 0);
}
/* Links is clicked or activated */
a:active {
	color: rgb(174, 0, 0);
}
```
